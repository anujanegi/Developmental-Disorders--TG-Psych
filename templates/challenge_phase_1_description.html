Data Description

File descriptions

  • X_train.csv - the training set features
  • X_test.csv - the test set features
  • y_train.csv - the training set labels
  • sample.csv - a sample submission file in the correct format
  • additional_X.csv - additional sample set features
  • additional_y.csv - additional sample set features labels

Data fields

Label

  • DX_01 - Diagnosis

Behavioral cognitive data

  • APQ_P_Total - Alabama Parenting Questionnaire (reported by parent)
  • APQ_SR_Total - Alabama Parenting Questionnaire (self reported)
  • ARI_P_Total_Score - Affective Reactivity Index (reported by parent)
  • ARI_S_Total_Score -Affective Reactivity Index (self reported)
  • Barratt_Total - Barratt Impulsiveness Scale
  • SCQ_Total - Social Communication Questionnaire
  • SWAN_Total - Rating Scale for ADHD
  • WISC_FSIQ - Wechsler Intelligence Scale for Children

Resting State EEG

Power Spectra
  • eyesclosed_psd_10dot50hz_average - power spectra density for 10.50 Hz frequency bin for eyes closed condition
  • eyesopen_psd_10dot50hz_average - power spectra density for 10.50 Hz frequency bin for eyes open condition
Microstates
  • eyesclosed_microstates_occurence_prototype1 - Occurence of a microstate (type 1) per sec
  • eyesclosed_microstates_coverage_prototype1 - % of time occupied by a microstate (type 1) (in the eyes closed condition)
  • eyesclosed_microstates_duration_prototype2 - Average duration of a microstate
  • eyesclosed_microstates_mspatcorr_prototype2 - Spatial Correlation between template maps and microstatesX_train.csv - the training set features
